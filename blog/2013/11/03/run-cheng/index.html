润成

一直在看《城市猎人》

一部悲剧得荡气回肠生生死死的爱恋。

几个主角都在绝望中希望，伤害中坚强却依旧善良纯净。

真彪无疑是让我最讨厌的，要不是他，润成不会为这场复仇承受这么多痛苦。

然而当他在那个冰冷的夜眼睁睁看着自己二十个战友为国荣征却被自己国家的兄弟亲手射杀。

他又怎么不是悲剧的受害者。

庆熙是可怜的，刚生完润成，丈夫就被自己国家的人杀死，儿子被真彪带走。

从此一个人悲伤的生活，只为能再见到儿子一面。

金娜娜的父亲惨遭车祸成植物人，她一直想为父亲雪冤，美丽，坚强。

润成见证过太多的厮杀，承受太多的痛苦，隐藏太多的情感，却始终一直正义而善良。

金英柱一身正气，誓坚守法律，却发现自己的父亲是最大的腐败者之一。

金英柱的父亲毁了金娜娜的一家。

有人说，金娜娜的父亲是当初射杀润成父亲的一员，后来因为想灭迹，所以金钟植企图将他一家撞死。

这又是怎样的悲情

三个人纠缠的上辈的恩怨情仇，下一辈之间的生死之恋。

这一场复仇中，没有谁对谁错，每一个人都是一个悲剧。

这也许就是悲剧最残忍的美。

美的那么让人心痛。

然而一切的源头都源于政治的黑暗，对权利的贪欲。

没有谁是胜者。

如果可以。

就让这两个人，一个帅的那么纯净，一个美的那么可人，让他们最简单地最自然的相恋相爱。

为生活中的一切琐事争吵，再更加确认的相爱，就这样。

该多好。

so goodbye, don’t cry and smile
